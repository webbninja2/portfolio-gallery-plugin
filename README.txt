== Description ==

This plugin will help you to display your portfolio with categories wise without page refresh, also it gave you the option to provide the detailed discription about your project/protfolio. Use this shortcode to display portfolio <strong>[ims-portfolio]</strong> 

* Check out the [  [Demo](http://web-xperts.xyz/demo/plugins/portfolio_gallery_ims/)  )

###⚡️ FEATURES

> * Simple using shortcodes
> * Portfolio discription and customize demo link lable
> * Portfolio like google image view
> * Gallary with filter
> * <strong>Fully Responsive</strong> 
> * <strong>Demo Content</strong>


== Installation ==

1. Go to the Plugins Menu in WordPress
2. Search for **" Portfolio Gallery Plugin "**
3. Click **" Install "**
4. Click **" Active "**
1. After activation the pluing use this short code **[ims-portfolio]** in wordpress backend.

== Frequently Asked Questions ==

= Q. What is the best image size for thumbnail?  =
You have to use 250x250 px image thumbnail for best view.

= Q. What is the best image size for full image?  =
For best view you have to use 1200x3000 px dimensions approx.

= Q. Where we can see the plugin demo? =
We have already set the demo content for you, you just need to put shortcode **[ims-portfolio]** in any page,post or template. 

= Q. Where we can add the images? =
Go to portfolieo  edit it and  upload the images.
